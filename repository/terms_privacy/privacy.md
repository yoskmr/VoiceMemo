# Soyoka プライバシーポリシー

## 1. はじめに

本プライバシーポリシー（以下「本ポリシー」）は、そよか運営チーム（以下「運営者」）が提供するiOSアプリケーション「Soyoka」（以下「本アプリ」）における、お客様の個人情報およびプライバシーに関する取り扱いについて説明するものです。

### アプリ概要

Soyokaは、音声録音とリアルタイム文字起こしを中心としたAI音声メモアプリです。録音した音声をテキストに変換し、AI技術による要約・タグ付け・感情分析などの機能を提供します。

### 運営者情報

- **運営者名**: そよか運営チーム
- **対応プラットフォーム**: iOS 17以降
- **連絡先**: 本ポリシー末尾の「お問い合わせ」をご参照ください

運営者は、お客様のプライバシーを尊重し、個人情報の保護に関する法律（個人情報保護法）をはじめとする関連法令を遵守いたします。本アプリをご利用いただく前に、本ポリシーの内容をよくお読みください。

| AI処理結果 | 要約テキスト、自動タグ、感情分析結果 | デバイス内のデータベース |

- お客様を個人として特定できる情報は一切収集しません
- IPアドレスは保存されません
- Cookieは使用しません
- 送信されるイベントは、録音の開始・完了、検索の実行、メモの閲覧・編集・削除、設定画面の表示、サブスクリプション画面の表示・購入の9種類に限定されます
- 収集されたデータからお客様個人を特定・追跡することはできません

### 2.3 収集しない情報

本アプリは、以下の情報を収集しません。

- 氏名、メールアドレス、電話番号などの連絡先情報（お問い合わせ時を除く）

- クロスアプリトラッキング（アプリ間でお客様を追跡するデータ）

## 3. 音声データの取り扱い

音声データはお客様の声を含むため、個人情報保護法上の「個人情報」に該当し得ます。本アプリでは、音声データの処理方法を「オンデバイス処理」と「クラウド処理」に明確に分離し、それぞれについて以下のとおり取り扱います。

### 3.1 オンデバイス処理（デバイス上で完結）

以下の処理は、すべてお客様のデバイス上で実行されます。音声データおよび文字起こし結果がデバイス外に送信されることはありません。

| 機能 | 使用技術 | データの送信先 |
|:----|:--------|:------------|
| 音声録音 | iOS標準API（AVFoundation） | なし（デバイス内保存） |
| リアルタイム文字起こし（Apple音声認識） | Apple Speech Recognition Framework | 下記参照 |
| リアルタイム文字起こし（SpeechAnalyzer） | SpeechAnalyzer（対応デバイスで利用可能） | **なし（完全オンデバイス）** |
| メモ保存・検索 | SwiftData / SQLite FTS5 | なし（デバイス内保存） |

**Apple音声認識について:**
Apple Speech Recognition Frameworkを使用する場合、本アプリはデフォルトでオンデバイス処理（`requiresOnDeviceRecognition = true`）を有効にしており、通常は音声データがデバイス外に送信されることはありません。ただし、オンデバイス認識が利用できない場合にApple側の仕様によりサーバー処理にフォールバックされる可能性があります。詳細はAppleのプライバシーポリシー（ https://www.apple.com/legal/privacy/ ）をご確認ください。

**SpeechAnalyzerについて:**
SpeechAnalyzerは、一部の新しいiOSバージョンで利用できるApple提供の完全オンデバイス音声認識フレームワークです。すべての処理がデバイス上で完結し、音声データが外部に送信されることはありません。マイク権限のみで動作し、音声認識権限の許可は不要です。

### 3.2 クラウド処理（外部サーバーを利用）

以下の機能では、テキストデータ（文字起こし済みのテキスト）を運営者が管理するサーバー（Backend Proxy）を経由して外部のAIサービスに送信します。**音声データ（音声ファイルそのもの）が外部に送信されることはありません。**

| 機能 | 送信されるデータ | 送信先 | 目的 |
|:----|:--------------|:------|:----|
| AI要約（クラウド） | 文字起こしテキスト（文字数の上限があります） | Backend Proxy → OpenAI API | テキストの要約生成 |
| 自動タグ付け（クラウド） | 文字起こしテキスト（文字数の上限があります） | Backend Proxy → OpenAI API | 内容に基づくタグ提案 |
| 感情分析 | 文字起こしテキスト（文字数の上限があります） | Backend Proxy → OpenAI API | テキストの感情傾向分析 |

**Backend Proxyについて:**
クラウドAI処理では、お客様のデバイスから直接OpenAI APIに接続するのではなく、運営者が管理するサーバー（Backend Proxy）を経由します。これにより、APIキーがお客様のデバイスに保存されることを防ぎ、通信の安全性を確保しています。Backend Proxyはデータの中継のみを行い、お客様のテキストデータを保存しません。

## 4. AI処理に関する方針

### 4.1 AI処理の基本方針

- AI処理に送信するデータは、**文字起こし済みのテキストのみ**です。音声ファイルは送信しません。
- AI処理（要約・タグ付け）は、録音完了後にバックグラウンドで自動実行されます。
- 感情分析はクラウド専用の機能であり、お客様がアプリ設定で明示的にオプトインした場合にのみ実行されます。
- AI処理の結果（要約、タグ、感情分析結果）は、デバイス内に保存されます。
- テキストの入力長には上限があります。上限を超える場合は分割して処理されます。

### 4.2 オンデバイスAI処理とクラウドAI処理

本アプリは、オンデバイスAI処理を優先する設計を採用しています。

| 処理方式 | 対象端末 | 使用技術 | データの送信先 |
|:--------|:--------|:--------|:------------|
| オンデバイス（優先） | 対応デバイス | Appleが提供するオンデバイスAIモデル | **なし（完全オンデバイス）** |
| クラウド（フォールバック） | 上記非対応端末、または感情分析利用時 | Backend Proxy → OpenAI API | 運営者が管理するサーバー（Backend Proxy） |

- **オンデバイスAI処理**: 対応デバイスでは、AI処理（要約・タグ付け）がデバイス上で完結します。テキストデータが外部に送信されることはありません。Appleが提供するオンデバイスAIモデルはiOSに内蔵されており、追加のダウンロードは不要です。
- **クラウドAI処理**: オンデバイス処理に非対応の端末では、Backend Proxyを経由してOpenAI が提供する大規模言語モデル（LLM）によるクラウド処理が実行されます。クラウドAI処理はProプラン（有料）の機能です。
- **感情分析**: 感情分析は常にクラウド処理で実行されます。お客様がアプリ設定で明示的にオプトインした場合にのみ有効になります。

### 4.3 利用するサードパーティAIサービス

| サービス名 | 提供元 | 利用目的 | プライバシーポリシー |
|:----------|:------|:--------|:-----------------|
| Appleが提供するオンデバイスAIモデル | Apple Inc. | オンデバイスでのテキスト要約・タグ付け | https://www.apple.com/legal/privacy/ |
| OpenAI が提供する大規模言語モデル（LLM） | OpenAI, Inc. | クラウドでのテキスト要約・タグ付け・感情分析 | https://openai.com/policies/privacy-policy |

> **注記:** OpenAI APIは、APIを通じて送信されたデータをモデルの学習に使用しない方針を採用しています（2024年3月以降のAPIデータ使用ポリシーに基づく）。ただし、この方針はOpenAI社の判断により変更される可能性があります。最新の状況はOpenAI社のプライバシーポリシーをご確認ください。

### 4.4 AI学習利用について

- 運営者は、お客様のデータ（音声、テキスト、メモ内容）を、AIモデルの学習・トレーニングに使用しません。
- サードパーティAIサービスにおけるデータの取り扱いは、各サービスのプライバシーポリシーに準じます。

## 7. データの保管・保護（安全管理措置）

個人情報保護法第23条に基づき、運営者は以下の安全管理措置を講じています。

### 7.1 技術的安全管理措置

- **デバイス内保存**: 音声データ・テキストデータは、デバイス内のデータベースに保存されます。iOSのデータ保護機能により、デバイスのロック時にはファイルが暗号化されます。
- **iCloudバックアップの除外**: 本アプリのデータ（音声ファイル・メモデータ）は、iCloudバックアップから明示的に除外されています。お客様がiCloudバックアップを有効にしている場合でも、本アプリのデータがiCloudにバックアップされることはありません。
- **通信の暗号化**: Backend Proxyおよびサードパーティサービスとの通信は、TLS 1.2以上による暗号化通信を使用します。
- **APIキーの保護**: サードパーティサービスのAPIキーは、運営者が管理するサーバー（Backend Proxy）を経由して使用され、お客様のデバイスにAPIキーが保存されることはありません。
- **デバイス認証**: Backend Proxyとの通信では、Appleが提供するデバイス認証技術によるデバイス認証を実施しています。これにより、正規のアプリからのリクエストのみが受け付けられ、不正なアクセスを防止します。

### 7.2 組織的安全管理措置

- 本アプリは個人開発であり、お客様の個人情報にアクセスできるのは運営者のみです。
- Backend Proxyはデータの中継のみを行い、お客様のテキストデータを保存しません。
- 運営者は、個人情報保護に関する法令・ガイドラインを遵守し、適切な取り扱いに努めます。

## 8. データの保持・削除

### 8.1 保持期間

- **デバイス内データ**: お客様が明示的に削除するか、アプリをアンインストールするまで、デバイス内に保持されます。運営者がデータの保持期間を制御することはありません。
- **AI処理時の送信データ**: Backend Proxyを経由してOpenAI APIに送信されたテキストデータは、Backend Proxyには保存されません。OpenAI APIにおけるデータ保持については、OpenAI社のデータ保持ポリシーに従います（2024年3月時点では、APIリクエストデータは最大30日間保持された後、削除されます）。
- **クラッシュログ・利用統計**: Appleのポリシーに準じます。
- **TelemetryDeckデータ**: TelemetryDeck GmbHのデータ保持ポリシーに従います。

### 8.2 削除方法

お客様は、以下の方法でデータを削除できます。

- **個別メモの削除**: アプリ内のメモ一覧画面からメモを選択し、削除操作を行ってください。音声データとテキストデータが完全に削除されます。
- **全データの削除**: アプリをアンインストールすることで、デバイス内のすべてのアプリデータが削除されます。

## 11. 海外ユーザーへの補足（GDPR / CCPA）

本アプリは日本国内向けに開発されていますが、App Storeを通じて海外からもご利用いただける場合があります。以下に、主要な海外プライバシー法制への対応を補足します。

### 11.1 EU一般データ保護規則（GDPR）への対応

EU/EEA地域にお住まいのお客様には、GDPRに基づく以下の権利が適用されます。

- **データの処理根拠**: お客様の同意、および本アプリのサービス提供に必要な範囲での契約の履行
- **データポータビリティの権利**: お客様のデータを機械可読形式で受け取る権利
- **処理の制限を求める権利**: 特定の状況下でデータ処理の制限を求める権利
- **異議申し立ての権利**: データ処理に対して異議を申し立てる権利
- **監督当局への苦情申し立て**: お住まいの国のデータ保護監督当局に苦情を申し立てる権利

お客様の個人データの越境移転が発生する場合（クラウドAI処理時）、適切な保護措置を講じます。

### 11.2 カリフォルニア州消費者プライバシー法（CCPA）への対応

カリフォルニア州にお住まいのお客様には、CCPAに基づく以下の権利が適用されます。

- **知る権利**: 収集される個人情報の種類と利用目的を知る権利
- **削除を求める権利**: 収集された個人情報の削除を求める権利
- **オプトアウトの権利**: 個人情報の「販売」を拒否する権利

**なお、運営者はお客様の個人情報を第三者に「販売」（CCPAの定義による）することはありません。**
