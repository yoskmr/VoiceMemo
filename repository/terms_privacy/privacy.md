## 1. はじめに

本プライバシーポリシー（以下「本ポリシー」）は、そよか運営チーム（以下「運営者」）が提供するiOSアプリケーション「Soyoka」（以下「本アプリ」）における、お客様の個人情報およびプライバシーに関する取り扱いについて説明するものです。

### アプリ概要

Soyokaは、音声録音とリアルタイム文字起こしを中心としたAI音声メモアプリです。録音した音声をテキストに変換し、AI技術による要約・タグ付け・感情分析などの機能を提供します。

### 運営者情報

- **運営者名**: そよか運営チーム
- **対応プラットフォーム**: iOS 17以降
- **連絡先**: 本ポリシー末尾の「お問い合わせ」をご参照ください

運営者は、お客様のプライバシーを尊重し、個人情報の保護に関する法律（個人情報保護法）をはじめとする関連法令を遵守いたします。本アプリをご利用いただく前に、本ポリシーの内容をよくお読みください。

## 2. 収集する情報

本アプリが収集・取り扱う情報は、以下のとおりです。

### 2.1 お客様が直接提供する情報

| 情報の種類 | 内容 | 保存場所 |
|:----------|:-----|:--------|
| 音声データ | 録音された音声ファイル | デバイス内（ローカル） |
| テキストデータ | 文字起こし結果、メモ本文、編集内容 | デバイス内（ローカル） |
| メモの付属情報 | タイトル、タグ、作成日時 | デバイス内（ローカル） |

### 2.2 自動的に生成・収集される情報

| 情報の種類 | 内容 | 保存場所 |
|:----------|:-----|:--------|
| AI処理結果 | 要約テキスト、自動タグ、感情分析結果 | デバイス内（ローカル） |
| アプリ利用分析 | 機能の利用状況、利用時間帯、セッション情報（匿名化済み、個人を特定不可） | TelemetryDeck のサーバー（EU） |
| アプリ利用情報 | クラッシュログ、利用統計（Apple提供の匿名化データ） | Apple のサーバー |

**アプリ利用分析（TelemetryDeck）について:**
本アプリは、サービス改善を目的として TelemetryDeck を利用した匿名の利用分析を行います。TelemetryDeck はプライバシーを重視した分析サービスであり、以下の特徴があります。

- お客様を個人として特定できる情報は一切収集しません
- IPアドレスは保存されません
- Cookieは使用しません
- 収集されるデータはデバイス上でハッシュ化された匿名識別子、アプリの利用状況（どの機能がいつ使われたか）、デバイスの種類・OSバージョンなどの一般的な技術情報のみです
- 収集されたデータからお客様個人を特定・追跡することはできません

### 2.3 収集しない情報

本アプリは、以下の情報を収集しません。

- 氏名、メールアドレス、電話番号などの連絡先情報（お問い合わせ時を除く）
- 位置情報
- 連絡先、カレンダー、写真ライブラリへのアクセス
- 広告識別子（IDFA）
- クロスアプリトラッキング（アプリ間でお客様を追跡するデータ）

## 3. 音声データの取り扱い

音声データはお客様の声を含むため、個人情報保護法上の「個人情報」に該当し得ます。本アプリでは、音声データの処理方法を「オンデバイス処理」と「クラウド処理」に明確に分離し、それぞれについて以下のとおり取り扱います。

### 3.1 オンデバイス処理（デバイス上で完結）

以下の処理は、すべてお客様のデバイス上で実行されます。音声データおよび文字起こし結果がデバイス外に送信されることはありません。

| 機能 | 使用技術 | データの送信先 |
|:----|:--------|:------------|
| 音声録音 | iOS標準API（AVFoundation） | なし（デバイス内保存） |
| リアルタイム文字起こし（WhisperKit） | WhisperKit（オンデバイスSTTエンジン） | **なし（完全オンデバイス）** |
| リアルタイム文字起こし（Apple音声認識） | Apple Speech Recognition Framework | 下記参照 |
| メモ保存・検索 | SwiftData / SQLite FTS5 | なし（デバイス内保存） |

**WhisperKitによる文字起こしについて:**
WhisperKitはデバイス上でAIモデルを実行するオンデバイスSTTエンジンです。音声データは一切デバイス外に送信されず、インターネット接続がない状態でも動作します。

**Apple音声認識について:**
Apple Speech Recognition Frameworkを使用する場合、Appleのプライバシーポリシーに基づき、音声データの一部がAppleのサーバーに送信される場合があります。詳細はAppleのプライバシーポリシー（https://www.apple.com/legal/privacy/ ）をご確認ください。お客様は設定画面からWhisperKit（完全オンデバイス）に切り替えることが可能です。

### 3.2 クラウド処理（外部サーバーを利用）

以下の機能では、テキストデータ（文字起こし済みのテキスト）を外部のAIサービスに送信します。**音声データ（音声ファイルそのもの）が外部に送信されることはありません。**

| 機能 | 送信されるデータ | 送信先 | 目的 |
|:----|:--------------|:------|:----|
| AI要約 | 文字起こしテキスト | OpenAI API | テキストの要約生成 |
| 自動タグ付け | 文字起こしテキスト | OpenAI API | 内容に基づくタグ提案 |
| 感情分析 | 文字起こしテキスト | OpenAI API | テキストの感情傾向分析 |

> **重要:** AI処理機能は将来のアップデートで提供予定です。提供開始時には、本ポリシーを更新し、利用するサードパーティサービスの詳細を明示します。AI処理の実行前には、アプリ内で個別にお客様の同意を取得します。

## 4. AI処理に関する方針

### 4.1 AI処理の基本方針

- AI処理に送信するデータは、**文字起こし済みのテキストのみ**です。音声ファイルは送信しません。
- AI処理は、お客様が明示的に機能を利用した場合にのみ実行されます。バックグラウンドでの自動送信は行いません。
- AI処理の結果（要約、タグ、感情分析結果）は、デバイス内に保存されます。

### 4.2 利用するサードパーティAIサービス

AI処理機能の提供開始時点で、以下のサービスの利用を予定しています。

| サービス名 | 提供元 | 利用目的 | プライバシーポリシー |
|:----------|:------|:--------|:-----------------|
| OpenAI API | OpenAI, Inc. | テキスト要約・タグ付け・感情分析 | https://openai.com/policies/privacy-policy |

> **注記:** OpenAI APIは、APIを通じて送信されたデータをモデルの学習に使用しない方針を採用しています（2024年3月以降のAPIデータ使用ポリシーに基づく）。ただし、この方針はOpenAI社の判断により変更される可能性があります。最新の状況はOpenAI社のプライバシーポリシーをご確認ください。

### 4.3 AI学習利用について

- 運営者は、お客様のデータ（音声、テキスト、メモ内容）を、AIモデルの学習・トレーニングに使用しません。
- サードパーティAIサービスにおけるデータの取り扱いは、各サービスのプライバシーポリシーに準じます。

## 7. データの保管・保護（安全管理措置）

個人情報保護法第23条に基づき、運営者は以下の安全管理措置を講じています。

### 7.1 技術的安全管理措置

- **デバイス内保存**: 音声データ・テキストデータは、SwiftData（SQLite）を用いてデバイス内に保存されます。iOSのデータ保護機能（Data Protection）により、デバイスのロック時にはファイルが暗号化されます。
- **通信の暗号化**: クラウドAIサービスとの通信は、TLS 1.2以上による暗号化通信を使用します。
- **APIキーの保護**: サードパーティサービスのAPIキーは、運営者が管理するバックエンドプロキシサーバーを経由して使用され、お客様のデバイスにAPIキーが保存されることはありません。

### 7.2 組織的安全管理措置

- 本アプリは個人開発であり、お客様の個人情報にアクセスできるのは運営者のみです。
- 運営者は、個人情報保護に関する法令・ガイドラインを遵守し、適切な取り扱いに努めます。

### 7.3 iCloudバックアップについて

お客様がiCloudバックアップを有効にしている場合、本アプリのデータ（音声ファイル・メモデータ）がiCloudにバックアップされる可能性があります。iCloudのデータ保護については、Appleのプライバシーポリシーをご確認ください。

## 8. データの保持・削除

### 8.1 保持期間

- **デバイス内データ**: お客様が明示的に削除するか、アプリをアンインストールするまで、デバイス内に保持されます。運営者がデータの保持期間を制御することはありません。
- **AI処理時の送信データ**: サードパーティAIサービスに送信されたテキストデータは、各サービスのデータ保持ポリシーに従います。OpenAI APIの場合、APIリクエストデータは最大30日間保持された後、削除されます（2024年3月時点のOpenAI社ポリシー）。
- **クラッシュログ・利用統計**: Appleのポリシーに準じます。

### 8.2 削除方法

お客様は、以下の方法でデータを削除できます。

- **個別メモの削除**: アプリ内のメモ一覧画面からメモを選択し、削除操作を行ってください。音声データとテキストデータが完全に削除されます。
- **全データの削除**: アプリをアンインストールすることで、デバイス内のすべてのアプリデータが削除されます。
- **iCloudバックアップからの削除**: iOSの設定 > Apple ID > iCloud > ストレージを管理 から、本アプリのバックアップデータを削除できます。

## 9. お客様の権利（開示・訂正・削除請求）

個人情報保護法に基づき、お客様は以下の権利を有します。

### 9.1 行使可能な権利

1. **利用目的の通知請求**: お客様の個人情報の利用目的について通知を求める権利
2. **開示請求**: 運営者が保有するお客様の個人情報の開示を求める権利
3. **訂正・追加・削除請求**: 個人情報の内容が事実でない場合に、訂正・追加・削除を求める権利
4. **利用停止・消去請求**: 個人情報が利用目的の範囲を超えて利用されている場合、または不正に取得された場合に、利用の停止・消去を求める権利
5. **第三者提供の停止請求**: 個人情報の第三者への提供の停止を求める権利

### 9.2 請求手続き

上記の権利を行使される場合は、本ポリシー末尾の連絡先までご連絡ください。ご本人確認をさせていただいたうえで、合理的な期間内に対応いたします。

本アプリのデータは原則としてお客様のデバイス内にのみ保存されており、運営者はこれにアクセスすることができません。そのため、デバイス内データの開示・訂正・削除は、お客様ご自身がアプリ内の操作で直接行うことができます。

## 11. 海外ユーザーへの補足（GDPR / CCPA）

本アプリは日本国内向けに開発されていますが、App Storeを通じて海外からもご利用いただける場合があります。以下に、主要な海外プライバシー法制への対応を補足します。

### 11.1 EU一般データ保護規則（GDPR）への対応

EU/EEA地域にお住まいのお客様には、GDPRに基づく以下の権利が適用されます。

- **データの処理根拠**: お客様の同意、および本アプリのサービス提供に必要な範囲での契約の履行
- **データポータビリティの権利**: お客様のデータを機械可読形式で受け取る権利
- **処理の制限を求める権利**: 特定の状況下でデータ処理の制限を求める権利
- **異議申し立ての権利**: データ処理に対して異議を申し立てる権利
- **監督当局への苦情申し立て**: お住まいの国のデータ保護監督当局に苦情を申し立てる権利

お客様の個人データの越境移転が発生する場合（AI処理時）、適切な保護措置を講じます。

### 11.2 カリフォルニア州消費者プライバシー法（CCPA）への対応

カリフォルニア州にお住まいのお客様には、CCPAに基づく以下の権利が適用されます。

- **知る権利**: 収集される個人情報の種類と利用目的を知る権利
- **削除を求める権利**: 収集された個人情報の削除を求める権利
- **オプトアウトの権利**: 個人情報の「販売」を拒否する権利

**なお、運営者はお客様の個人情報を第三者に「販売」（CCPAの定義による）することはありません。**
