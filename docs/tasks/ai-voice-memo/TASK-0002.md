## 実装詳細

### 1. VoiceMemo @Model :large_blue_circle:

01-Archセクション5.2準拠。統合仕様書のAudioFormat(.m4a)を使用。

```swift
// Domain/Models/VoiceMemo.swift
@Model
final class VoiceMemo {
    @Attribute(.unique) var id: UUID
    var title: String
    var createdAt: Date
    var updatedAt: Date
    var durationSeconds: Double
    var audioFilePath: String    // Documents/Audio/ からの相対パス
    var audioFormat: AudioFormat
    var status: RecordingStatus
    var isFavorite: Bool

    @Relationship(deleteRule: .cascade, inverse: \Transcription.memo)
    var transcription: Transcription?

    @Relationship(deleteRule: .cascade, inverse: \AISummary.memo)
    var aiSummary: AISummary?

    @Relationship(deleteRule: .cascade, inverse: \EmotionAnalysis.memo)
    var emotionAnalysis: EmotionAnalysis?

    @Relationship(deleteRule: .nullify)
    var tags: [Tag]
}

enum AudioFormat: String, Codable { case m4a, opus }
enum RecordingStatus: String, Codable { case recording, paused, completed, failed }
```

### 5. ModelContainer構成 :large_blue_circle:

01-Archセクション6.3準拠。SecureStore/ディレクトリにSwiftDataストアを配置。

```swift
// InfraStorage/SwiftDataStore/ModelContainerConfiguration.swift
enum ModelContainerConfiguration {
    static func create() throws -> ModelContainer {
        let schema = Schema([
            VoiceMemo.self, Transcription.self, AISummary.self,
            Tag.self, EmotionAnalysis.self, UserSettings.self,
        ])
        let configuration = ModelConfiguration("VoiceMemoStore", schema: schema)
        let container = try ModelContainer(for: schema, configurations: [configuration])
        excludeSwiftDataStoreFromBackup()
        return container
    }
}
```

### 6. Data Protection + iCloudバックアップ除外 :large_blue_circle:

統合仕様書セクション8準拠。
