## タスク概要

Apple Speech Framework（SFSpeechRecognizer）を用いたリアルタイムストリーミング音声認識エンジンを実装する。統合仕様書セクション3.1準拠のSTTEngineProtocol（AsyncStreamベース）に準拠する。iOS 17-25ではSFSpeechRecognizerを、iOS 26+ではSpeechAnalyzerを使用する分岐を含む。日本語言語パック対応、オフライン認識モードをサポートする。

## 実装詳細

### 1. AppleSpeechEngine :large_blue_circle:

統合仕様書セクション3.1のSTTEngineProtocol準拠。callbacks方式は使用しない。

```swift
// InfraSTT/Engines/AppleSpeechEngine.swift
import Speech
import AVFoundation

final class AppleSpeechEngine: STTEngineProtocol, @unchecked Sendable {
    let engineType: STTEngineType = .speechAnalyzer

    private let recognizer: SFSpeechRecognizer
    private var recognitionRequest: SFSpeechAudioBufferRecognitionRequest?
    private var recognitionTask: SFSpeechRecognitionTask?

    init(locale: Locale = Locale(identifier: "ja-JP")) {
        self.recognizer = SFSpeechRecognizer(locale: locale)!
    }

    var supportedLanguages: [String] {
        SFSpeechRecognizer.supportedLocales().map { $0.identifier }
    }

    func startTranscription(
        audioStream: AsyncStream<AVAudioPCMBuffer>,
        language: String
    ) -> AsyncStream<TranscriptionResult> {
        AsyncStream { continuation in
            let request = SFSpeechAudioBufferRecognitionRequest()
            request.shouldReportPartialResults = true
            request.requiresOnDeviceRecognition = true  // オフライン優先

            self.recognitionRequest = request

            // audioStreamからPCMバッファをSpeech Frameworkに供給
            Task {
                for await buffer in audioStream {
                    request.append(buffer)
                }
                request.endAudio()
            }

            // 認識結果をAsyncStreamに変換
            self.recognitionTask = recognizer.recognitionTask(with: request) { result, error in
                if let result = result {
                    let transcriptionResult = TranscriptionResult(
                        text: result.bestTranscription.formattedString,
                        confidence: Double(result.bestTranscription.segments.last?.confidence ?? 0),
                        isFinal: result.isFinal,
                        language: language,
                        segments: result.bestTranscription.segments.map { segment in
                            TranscriptionSegment(
                                text: segment.substring,
                                startTime: segment.timestamp,
                                endTime: segment.timestamp + segment.duration,
                                confidence: Double(segment.confidence)
                            )
                        }
                    )
                    continuation.yield(transcriptionResult)

                    if result.isFinal {
                        continuation.finish()
                    }
                }
                if let error = error {
                    continuation.finish()
                }
            }

            continuation.onTermination = { @Sendable _ in
                self.recognitionRequest?.endAudio()
                self.recognitionTask?.cancel()
            }
        }
    }

    func finishTranscription() async throws -> TranscriptionResult {
        recognitionRequest?.endAudio()
        // 最終結果を待つ
        // ...
    }

    func isAvailable() async -> Bool {
        guard recognizer.isAvailable else { return false }
        let status = SFSpeechRecognizer.authorizationStatus()
        return status == .authorized
    }

    func setCustomDictionary(_ dictionary: [String: String]) async {
        // SFSpeechRecognizer にはカスタム辞書APIなし
        // contextualStrings で部分的に対応
        if let request = recognitionRequest {
            request.contextualStrings = Array(dictionary.values)
        }
    }
}
```

### 2. 権限リクエスト :large_blue_circle:

```swift
static func requestAuthorization() async -> Bool {
    await withCheckedContinuation { continuation in
        SFSpeechRecognizer.requestAuthorization { status in
            continuation.resume(returning: status == .authorized)
        }
    }
}
```

### 3. オフライン認識モード :large_blue_circle:

`requiresOnDeviceRecognition = true` を設定し、ネットワーク不要のオンデバイス認識を有効化。iOS 17+ではSFSpeechRecognizerのオンデバイス認識が日本語をサポートしている。

```swift
request.requiresOnDeviceRecognition = true
// オンデバイス認識がサポートされていない場合のフォールバック
if !recognizer.supportsOnDeviceRecognition {
    request.requiresOnDeviceRecognition = false  // サーバー認識にフォールバック
}
```

## テスト要件

### 正常系
- `startTranscription` でAsyncStreamが返却され、部分結果が流れること
- `isFinal = true` の結果が最終的に返されること
- `finishTranscription()` で最終確定結果が取得できること
- 日本語音声の認識が正しく動作すること（テスト用の音声バッファを使用）
- `isAvailable()` が権限許可済み時にtrueを返すこと

### 異常系
- 権限未許可時に `isAvailable()` がfalseを返すこと
- SFSpeechRecognizerが利用不可の場合のエラーハンドリング
- 認識タスクのキャンセルが正しく動作すること

### オフライン
- `requiresOnDeviceRecognition = true` でオフライン認識が動作すること
- オンデバイス認識非サポート環境でのフォールバック確認

### AsyncStream
- AsyncStreamが正常に終了（finish）されること
- onTermination時にリソースが解放されること
- 複数のyieldが順序通り配信されること
