### 4. プライバシーポリシー :large_blue_circle:

```
プライバシーポリシー要点:
- 収集データ: なし（全てローカル保存）
- クラウド送信: Proプランのクラウド処理時のみ、音声/テキストデータを一時的に送信
```

### 5. Info.plist権限文言 :large_blue_circle:

App Review対応。権限の使用目的を明確に記述。

```xml
<!-- Info.plist -->

<!-- マイク権限 -->
<key>NSMicrophoneUsageDescription</key>
<string>音声メモを録音するためにマイクへのアクセスが必要です。録音データはこのデバイスにのみ保存されます。</string>

<!-- 音声認識権限 -->
<key>NSSpeechRecognitionUsageDescription</key>
<string>録音した音声をリアルタイムでテキストに変換するために音声認識を使用します。処理はデバイス上で行われます。</string>
```

**英語版（InfoPlist.strings）**:
```
"NSMicrophoneUsageDescription" = "Microphone access is required to record voice memos. Recording data is stored only on this device.";
"NSSpeechRecognitionUsageDescription" = "Speech recognition is used to convert recorded audio to text in real-time. Processing is performed on the device.";
```

### 6. App Review対応ノート :large_blue_circle:

```
App Review Notes:

1. マイク・音声認識権限について
   本アプリは音声メモの録音と文字起こしのためにマイクと音声認識権限を使用します。
   録音データはデバイス内にのみ保存され、ユーザーの明示的な操作なしにクラウドへ
   送信されることはありません。

2. AI処理について
   - オンデバイス処理: WhisperKit（文字起こし）、llama.cpp（要約・タグ付け）を
     デバイス上で実行します。ネットワーク通信は発生しません。
   - クラウド処理（Proプラン）: ユーザーがProプランに加入し、かつネットワーク接続が
     ある場合のみ、Cloudflare Workers経由でLLM APIに一時的にテキストデータを送信します。
     サーバーにデータは保持されません。

3. サブスクリプションについて
   - 月額プラン: ¥500/月（自動更新）
   - 年額プラン: ¥4,800/年（自動更新、2ヶ月分お得）
   - 解約: 設定 > Apple ID > サブスクリプションからいつでもキャンセル可能

4. デモアカウント
   本アプリにログイン機能はありません。インストール後すぐにご利用いただけます。
   Proプラン機能を確認する場合は、課金画面の「プランを始める」ボタンから
   サブスクリプションをお試しください。

5. バックエンドサーバー
   Cloudflare Workers上にLLM Proxyサーバーを運用しています。
   URL: https://api.voicememo.app
   App Attestによるデバイス認証を実施しています。
```

## 実装手順
1. **direct-setup**: App Store Connect アプリ登録 -> メタデータ入力 -> スクリーンショット作成・アップロード -> 説明文入力 -> キーワード設定

3. **direct-review**: Info.plist権限文言設定 -> App Review対応ノート作成 -> TestFlight設定
4. **direct-verify**: 全設定項目の確認 -> TestFlight内部テストビルド配布 -> 動作確認
