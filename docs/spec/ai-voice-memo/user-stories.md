## 優先度とMoSCoWマッピング

### MoSCoW分類の定義

| 分類 | 定義 |
|:-----|:-----|
| **Must Have** | MVPリリースに必須。これがないとアプリとして成立しない機能 |
| **Should Have** | MVPに含めたいが、なくても最低限動作する機能 |
| **Could Have** | あると良いが、リリース後の追加でも問題ない機能 |
| **Won't Have (this time)** | 今回のスコープ外。将来バージョンで検討 |

### MoSCoW ↔ 優先度マッピング

| MoSCoW分類 | 対応する優先度 | 対象エピック |
|:-----------|:-------------|:------------|
| Must Have | P1（録音+STT） | エピック1: 音声録音・文字起こし |
| Should Have | P2（メモ管理+検索） | エピック2: メモ管理・検索 |
| Could Have | P3-P4（AI要約・感情分析・エクスポート・UI） | エピック3〜5 |
