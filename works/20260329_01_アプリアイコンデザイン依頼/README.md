# Soyoka アプリアイコン デザイン依頼書

**作成日**: 2026-03-29
**依頼元**: Soyoka 開発チーム

---

## 1. アプリ概要

| 項目 | 内容 |
|------|------|
| **アプリ名** | Soyoka（そよか） |
| **名前の由来** | 「そよ風」+「香」。穏やかな風が香るイメージ |
| **カテゴリ** | Productivity / 音声メモ |
| **プラットフォーム** | iOS 17+ (iPhone / iPad) |
| **サブタイトル** | 声でつぶやくだけ。AIがあなたの言葉を整えます。 |
| **ターゲットユーザー** | 20〜40代、思いつきや日記を声で残したい人 |

### アプリの一言説明

> 声のままでいい。ちゃんと残るから。

話しかけるだけで、声がきちんとしたメモになる音声メモアプリ。散歩中のアイデア、家事の合間のつぶやき、日記的な記録など、**個人の思考整理**に特化。

---

## 2. ブランドアイデンティティ

### デザイン3原則

| 原則 | 説明 |
|------|------|
| **即時性（Immediacy）** | 思いついた瞬間にすぐ使える。UIは思考を妨げない |
| **温かみ（Warmth）** | 機械的でなく、手書きの日記帳を開くような親しみやすさ |
| **プライバシー安心感（Privacy Assurance）** | 個人的な思考を預かるアプリとしての信頼感 |

### ブランドトーン

- **やわらかく、おだやか**（テック寄りでない）
- **親しみやすく、あたたかい**（高級感より安心感）
- **シンプルで静か**（派手さより落ち着き）
- 「そよ風」の軽やかさ、「つぶやき」を受けとめるやさしさ

### ポジショニング

「会議議事録ツール」ではなく「個人の思考整理ツール」。
競合の多くがテック感・ビジネス寄りの中、Soyokaは日記やひとりごとを受けとめる**パーソナルで温かいアプリ**として差別化。

### カラーの方針

- 角丸12pt以上でやわらかい印象
- 丸ゴシック系フォント使用

> **アイコンはこのカラーシステムと調和すること。ただし、アイコン単体での視認性・印象を優先してOK。**

## 4. アイコンに表現してほしいこと

### コアモチーフの候補（参考。デザイナーの提案歓迎）

| モチーフ | 意図 |
|----------|------|
| **そよ風 / 風の流れ** | アプリ名の由来「そよか」。やわらかさ、自然の心地よさ |
| **声 / 音の波形** | 音声メモアプリとしての機能を示唆 |
| **メモ / ノート** | 記録を残すというコア体験 |
| **吹き出し** | 「つぶやき」を受けとめるイメージ |
| **葉っぱ / 自然モチーフ** | オーガニックな温かみ、デバイス内処理（プライバシー）|

### 表現してほしいニュアンス

- 「話しかけたくなる」やさしさ
- テクノロジーを感じさせすぎない自然体な印象
- 軽やか・風通しの良さ（重厚感は不要）
- 個人的で親密な感じ（ビジネスツール感ではない）

### 避けてほしいこと

- マイクのアイコンそのもの（競合と差別化できない）
- ギアやロボットなどテック感の強いモチーフ
- 鮮やかすぎるネオンカラー
- 直線的・幾何学的なデザイン
- 3D/リアリスティックな影表現

## 5. 競合アプリのアイコン参考

App Storeで並ぶ際の差別化を意識すること。

| アプリ | アイコンの特徴 | Soyokaとの差別化ポイント |
|--------|--------------|------------------------|
| **Apple ボイスメモ** | 波形 + 暗い背景 | → Soyokaは暖色・やわらかさで差別化 |
| **Otter.ai** | 青系 + カワウソ | → Soyokaはビジネス感を出さない |
| **Notta** | 紫 + N文字 | → Soyokaは文字ロゴではなくモチーフ重視 |
| **muute** | ミニマル + モノクロ | → Soyokaは温かみの色を使う |
| **CLOVA Note** | 緑 + ノート | → Soyokaはより個人的・パーソナルな印象 |
| **Whisper Notes** | マイクアイコン | → Soyokaはマイク直球を避ける |

**目標**: 「音声メモ」カテゴリの検索結果で並んだとき、温かみ・やさしさで目を引く存在。

- テキストは最小限（入れない方が望ましい）

## 10. 補足情報

### アプリ名の背景

「Soyoka」は日本語の「そよか」。造語で、「そよ風（gentle breeze）」と「香（fragrance）」を組み合わせた名前。穏やかな風が香りを運ぶように、ユーザーの声をやさしく受けとめて整えるアプリ。

### 主な機能

1. **ワンタップ録音** — 画面中央のボタンですぐ開始
2. **リアルタイム文字起こし** — 話しながらテキスト化
3. **AI整理** — つぶやきを読みやすく整える
4. **感情の記録** — 声のトーンから気持ちを記録
5. **全文検索** — 過去のメモをキーワードで検索

### プライバシーへのこだわり

- 音声データはデバイスにのみ保存
- 音声認識はデバイス上で処理
- 第三者への提供は一切なし
