# 20260315_05_日本市場競合再調査

## 依頼内容

前回調査（2026-03-14）で「日本語 x AI音声日記は明確な空白地帯」と結論付けたが、本当に空白なのかを再検証する。日本市場に特化した競合調査を実施。

## 実施した作業内容

1. **日本発の音声メモ・音声日記アプリの網羅的調査**
   - App Store / Google Play のランキング調査
   - 個人開発・スタートアップアプリの発掘（シャべマル、声log、こえにっき、日記AI、マイモリ等）

2. **グローバルアプリの日本語対応状況調査**
   - Notta、Murmur（ひそひそ）、Texter、Voicenotes、Untold、YoByte

3. **AIジャーナリング隣接市場の調査**（前回未調査）
   - muute（100万DL）、Awarefy（80万DL）、hinoto（2026年1月リリース）

4. **各競合のAI機能・感情分析・オンデバイス処理・価格の評価**

## 得られた知見・結果

### 前回結論の修正

| 前回 | 今回 |
|---|---|
| 完全な空白 | **条件付きの部分的空白** |
| 直接競合なし | **Murmur、シャべマル、マイモリが存在** |
| 隣接脅威なし | **muute(100万DL)、Awarefy(80万DL)が需要実証** |

### 新発見

- シャべマル: Whisper+ChatGPTの日本語音声メモ日記（個人開発、月額500円）
- 日記AI: 同コンセプトアプリがラッコM&Aで10万円売却（収益化失敗の反面教訓）
- muute(100万DL): AI x 感情分析 x 日記の需要が日本で実証済み（テキストのみ）
- 音声日記ジャンル自体が日本で未成熟（ランキング上位に音声日記なし）
