# 競合ビジネスモデル・価格設定調査

## 依頼内容

音声メモアプリ競合9社のビジネスモデル（価格設定）を調査。以下の項目を重点的に分析:

- 無料プランで何ができるか（制限事項）
- 有料プランの価格と提供機能
- LLM/AI機能は無料と有料のどちらに含まれるか
- 買い切りアプリはAI機能をどう提供しているか（APIコストをどう吸収しているか）
- フリーミアムの無料→有料転換率

## 実施した作業

1. WebSearchによる9アプリの価格・機能情報の収集
2. WebFetchによる公式価格ページの詳細確認
3. フリーミアムSaaS転換率の業界ベンチマーク調査
4. 買い切りアプリのLLMコスト持続可能性の定量分析
5. ビジネスモデル4類型への分類と比較分析

## 主要な知見

1. **買い切り + クラウドLLM は構造的に不可能**: $4.99の買い切りアプリがクラウドLLMを使う場合、5.5ヶ月で売上を超過し赤字化
2. **現存の買い切りアプリはLLMを使っていない**: Just Press Record（Siri）とWhisper Notes（オンデバイスWhisper）はサーバーコストゼロで成立
3. **$8-10/月が市場のアンカー価格**: Otter.ai, Notta, Voicenotes, VOMO, AudioPenがこの価格帯に集中
4. **SaaS フリーミアム転換率は2-5%**: Otter.aiは推定約4%で、ARR $100Mを達成
5. **Lifetime販売はリスクを伴う**: Audionotes $199は18ヶ月で損益分岐、Voicenotes $50はさらに早期
