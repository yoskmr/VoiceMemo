# 新興AI音声メモ・文字起こしアプリ 競合調査レポート (2025-2026)

**調査日**: 2026-03-30
**目的**: Soyoka（個人の思考整理ツール）の直接・間接競合となる新興AIアプリの把握

---

## 1. 調査サマリー

2025-2026年にかけて、AI音声メモ市場は急速に拡大している。特に以下の3つのトレンドが顕著:

1. **GPT/Claude連携の標準化**: 単なる文字起こしから「AIと対話できるノート」への進化（Voicenotes, VOMO AI）
2. **思考整理・ジャーナリング特化の台頭**: 会議議事録とは別に、個人の振り返り・アイデア整理に特化するアプリが増加（AudioPen, Cleft, Voxly+）
3. **感情分析の実用化**: 音声のトーン・リズムからの感情検出が85-90%精度に到達（Voxly+, Untold）

## 2. 主要アプリ詳細分析

### 2.1 VOMO AI

| 項目 | 内容 |
|:-----|:-----|
| **公式サイト** | [vomo.ai](https://vomo.ai) |
| **プラットフォーム** | iOS / Web |
| **リリース時期** | 2024年頃、2025-2026年に急成長 |
| **主要AI機能** | Whisper + GPT-4 Turbo による文字起こし・要約・翻訳、インタラクティブトランスクリプトチャット |
| **対応言語** | 50言語以上（日本語専用チューニングモデルあり） |
| **精度** | 98%（公称） |
| **価格** | 無料（30分）/ Pro $9.99/月 / 年額 $71.88（7日間無料トライアル） |
| **ターゲット** | フリーランス、学生、小規模チーム |
| **特徴的機能** | YouTube動画の文字起こし、トランスクリプトへのAIチャット、50+言語翻訳 |

**Soyokaとの競合度: 高**

- 日本語専用チューニングモデルを持つ点が脅威
- GPT-4oチャット機能でAI連携が成熟
- ただし「会議・講義」寄りのポジショニングで、思考整理特化ではない
- サーバー依存のため、オフライン利用はSoyokaが優位

### 2.2 AudioPen

| 項目 | 内容 |
|:-----|:-----|
| **公式サイト** | [audiopen.ai](https://www.audiopen.ai/) |
| **プラットフォーム** | iOS / Android / Web / Chrome拡張 |
| **リリース時期** | 2023年、2025年に大幅成長 |

| **ターゲット** | ライター、クリエイター、思考を文章化したい個人 |
| **特徴的機能** | 「とりとめのない話を整った文章にする」ことに特化、文体カスタマイズ |

**Soyokaとの競合度: 高**

## 3. 競合マップ（ポジショニング）

```
                    思考整理・ジャーナリング
                           ↑
                           |
          AudioPen ●       | ● Voicenotes
                           |
    Cleft ●                | ● Voxly+
                           |
    Reflect ●              |
  ─────────────────────────┼─────────────────────── オンデバイス ← → クラウド
                           |
    Spokenly ●             | ● VOMO AI
                           |
                           | ● Oasis AI
    Soyoka ★               |
    (目標ポジション)        | ● Talknotes
                           |
                           | ● Audionotes
                           ↓
                    ビジネス・会議・コンテンツ制作
```

**Soyokaの目標ポジション**: 左上象限（オンデバイス × 思考整理）
- この象限で最も近いのはCleftとAudioPen
- ただし、AudioPenはクラウド依存、Cleftは構造化ノート寄り
- **「オンデバイスAI × 日本語最適化 × 思考整理特化」の組み合わせは現時点で空白**

## 5. Soyokaへの戦略示唆

### 5.1 差別化を強化すべきポイント

1. **オンデバイスAI × 日本語最適化**: この組み合わせは競合に空白。SpeechAnalyzer + WhisperKitの二重エンジンは独自価値
2. **感情分析の早期実装**: Voxly+が先行しているが、日本語音声の感情分析はまだ空白
3. **FTS5全文検索**: 蓄積されたメモの高速検索はVoicenotes/AudioPenにない強み

### 5.2 競合から学ぶべき機能

| 機能 | 参考アプリ | 優先度 |
|:-----|:---------|:------|
| Ask AI（蓄積ノートへのAIチャット） | Voicenotes | 高 |
| SuperSummaries（複数ノート横断要約） | AudioPen | 高 |
| 出力テンプレート（ブログ/メール/TODO等） | Oasis AI, Talknotes | 中 |
| ナレッジグラフ（ノート間の関連自動検出） | Reflect Notes | 中 |
| 感情トラッキング可視化 | Voxly+ | 中 |
| Apple Watch対応 | Voicenotes | 低（Phase 4以降） |

### 5.3 価格戦略の参考

| アプリ | 無料枠 | 有料プラン |
|:------|:------|:---------|
| VOMO AI | 30分 | $9.99/月, $71.88/年 |
| AudioPen | 3分/10ノート | $75-99/年 |
| Voicenotes | 制限付き | $10/月, $100/年 |
| Spokenly | オフライン無料 | $9.99/月 |
| Cleft | 5分録音制限 | $7/月 |
| Oasis AI | 30クレジット | $5/月, $50/年 |
| **Soyoka（推奨）** | **オンデバイス基本機能無料** | **$5-7/月（AI高度機能）** |

Soyokaはオンデバイス処理によりサーバーコストが低いため、**競合より低価格でのサブスク**、または**買い切りモデル**が可能。これは大きな競争優位。

### 5.4 ウォッチリスト（継続監視対象）

- **AudioPen**: UXの進化速度が速く、思考整理市場のリーダー候補
- **Voicenotes**: Ask AI機能の発展とApple Watch対応
- **Voxly+**: 感情分析の精度向上と市場浸透度
- **Apple ボイスメモ (iOS 26)**: SpeechAnalyzer搭載済み、さらなるAI機能追加の可能性
- **NVIDIA Parakeet v4以降**: Whisper代替エンジンの進化
