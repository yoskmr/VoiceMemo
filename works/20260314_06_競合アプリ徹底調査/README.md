# 20260314_06_競合アプリ徹底調査

## 依頼内容

音声入力を使った日記・メモアプリのグローバル市場における主要競合アプリの徹底調査。単なるボイスレコーダーではなく、音声をテキストに変換して日記やメモとして活用するアプリに焦点を当てた競合分析。

## 実施した作業内容

- 5カテゴリ + 1新興カテゴリ（ウェアラブル）に分けて計28プロダクトを調査
- 各アプリの主要機能、差別化ポイント、価格、ユーザー規模、技術スタック等を整理
- ポジショニングマップ（mermaid）を作成
- 市場トレンド、空白地帯、価格帯分析を実施

## 得られた知見

1. **オンデバイス処理（プライバシー重視）** がトレンド化
2. **ウェアラブル化** が加速（Plaud 100万台、Meta/AmazonのM&A）
3. **日本語×AI音声日記** は明確な空白地帯
4. **サブスク疲れ** に対する買い切りモデルの再評価
5. **感情AI統合** が日記カテゴリの差別化要因に
