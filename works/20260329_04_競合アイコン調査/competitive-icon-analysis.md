## 2. 色分布マップ分析

### 2-1. 色の使用頻度

```
青・紺系    ████████████  (7アプリ) Otter, Notta(サブ), Day One, Craft, Group Transcribe, Speechy, Whisper Notes
ダーク/黒系 ████████      (5アプリ) Apple VM, Notta, Notion, Texter, Whisper Notes
赤/ピンク系 ██████        (4アプリ) Notta, Apple VM, Bear, muute(パステル)
緑系        ████          (3アプリ) CLOVA Note, Journey, Apple メモ(サブ)
黄系        ████          (2アプリ) Apple メモ, Day One(星)
紫系        ████          (3アプリ) VOMO, muute(ラベンダー), Notta(サブ)
オレンジ系  ██            (1アプリ) Penzu
白系        ████          (3アプリ) Notion, Bear, Apple メモ
パステル系  ██            (1アプリ) muute
```

### 2-2. 色の傾向分析

1. **青系が圧倒的に多い**: 全17アプリ中7アプリ（41%）が青を主要色として使用。「信頼性」「テクノロジー」「落ち着き」の連想。音声・文字起こし系は特に青への偏りが顕著
2. **ダーク背景が増加傾向**: Apple ボイスメモ（iOS 12で白→黒に変更）、Notta、Whisper Notes、Texter など。プロフェッショナル感の演出
3. **暖色系は極めて少ない**: オレンジ主体は Penzu のみ。ピーチ/テラコッタ系は皆無
4. **パステル系はmuuteのみ**: ジャーナル系でもパステルを全面に出しているのは muute だけ

## 3. モチーフ傾向分析

### 3-1. モチーフ使用頻度

```
音声波形      ████████  (4アプリ) Apple VM, Whisper Notes, CLOVA Note, Speechy
マイク        ██████    (3アプリ) VOMO, Speechy, CLOVA Note
文字/テキスト ██████    (3アプリ) Notion, Texter, Notta
動物/マスコット██        (2アプリ) Otter(カワウソ), Bear(クマ)
ペン/書く道具 ████      (2アプリ) Penzu, Apple メモ
自然モチーフ  ████      (2アプリ) Journey(葉), muute(有機的形状)
幾何学/抽象  ████      (3アプリ) Craft, Day One, muute
人物/グループ ██        (1アプリ) Group Transcribe
```

### 3-2. モチーフの傾向

1. **音声波形 + マイク**: 音声系アプリの定番。差別化が困難
2. **文字ロゴ（N, C等）**: Notion、Notta、Craft がこのアプローチ。ブランド名の頭文字
3. **動物マスコット**: Otter.ai が最も成功。Bear もクマで確立。感情移入しやすい
4. **自然・有機的モチーフ**: Journey（葉）、muute（抽象有機体）。ウェルビーイング系に多い
5. **「風」「そよ風」モチーフは皆無**: 調査対象17アプリに風をモチーフにしたものは存在しない

| モチーフ | 使用状況 | 空白度 |
|:---------|:---------|:-------|

### 4-3. 最大の空白ゾーン（色 x モチーフ）

```
暖色（ピーチ/テラコッタ） x 自然モチーフ（風/花/香り）
 → 競合ゼロ。Soyokaが唯一の占有者になれる

暖色 x 有機的・柔らかい形状
 → muuteがパステルで近いが、暖色ではない

暖色 x ミニマルな文字ロゴ
 → Notion(黒), Notta(ピンク寄り), Craft(青)はあるが、暖色系は空白
```

## 5. Soyokaへの提言

### 5-1. 推奨ポジション: 「暖色 x 自然（風）モチーフ」の独占

Soyokaのブランドカラー (#E8956A) は、競合17アプリの中で**完全な空白ゾーン**に位置する。この暖色ポジションは、以下の理由から最適である:

**App Store 棚での視認性**
- 音声メモ/文字起こしカテゴリは青・ダーク系で埋め尽くされている
- ピーチ/テラコッタ系の暖色は、検索結果一覧で**即座に目立つ**
- 2025-2026年のデザイントレンドでも「暖色でより自然で感情的なデザイン」が台頭中

**ブランドとの整合性**
- 「そよか（そよ風 + 香）」のネーミングと暖色は自然な連想
- 個人の思考整理ツールという温かみのあるポジションに合致
- プライバシー安心感 = 「家」「暖炉」の連想 = 暖色

**muuteとの差別化**
- muuteはパステル（淡い/冷たい印象）、Soyokaはテラコッタ（温かい/地に足ついた印象）
- muuteは抽象的、Soyokaはより具体的なモチーフで差別化可能

### 5-2. モチーフ提案

#### A案: そよ風モチーフ（最推奨）
- 柔らかい曲線で風の流れを表現
- 音声波形を風の流れに見立てるダブルミーニング（音声 + そよ風）
- 花びらが風に舞う要素を微かに加える
- **利点**: 競合ゼロのモチーフ。Soyokaの名前との直接的な連想

#### B案: 花びら/香りモチーフ
- 花びらの柔らかい形状をアイコンの中心に
- 「香（か）」の部分を視覚化
- **利点**: 女性ユーザーへの訴求力。ウェルビーイング連想

#### C案: 抽象的な暖色グラデーション + ミニマル「S」
- Craft (#3555FF の C) やNotion (黒の N) の暖色版アプローチ
- 「S」の文字を風の流れのように柔らかく造形
- **利点**: テック系の洗練さと暖色の温かみの両立

### 5-3. 避けるべきデザイン要素

| 避けるべき要素 | 理由 |
|:---------------|:-----|
| **マイクアイコン** | VOMO, Speechy, CLOVA Note と競合。最も汎用的で差別化不可 |
| **音声波形（直線的）** | Apple ボイスメモ、Whisper Notes と直接競合 |
| **青/紺を主要色に** | 7アプリと被る。最も混雑した色域 |
| **ダーク背景** | Apple VM, Notta, Texter, Notion と差別化できない |
| **文字ロゴのみ** | Notion, Notta, Craft の後追いに見える |
| **動物マスコット** | Otter.ai, Bear の模倣に見える |

### 5-4. iOS 26 Liquid Glass への対応

Apple の新デザイン言語「Liquid Glass」（2025 WWDC発表）への対応も考慮が必要:

- **多層構成**: Icon Composer で前景/中景/背景の3レイヤー設計
- **半透明効果との相性**: テラコッタ/ピーチ系の暖色は Liquid Glass の半透明効果と相性が良い（青系は背景に溶け込みやすいが、暖色は際立つ）
- **ダーク/ライト/ティンテッド3モード対応**: iOS 18+ で必須。暖色はティンテッドモードでも識別性が高い

### 5-5. 推奨デザインブリーフ

```

補助色:       クリーム〜ウォームベージュ
```

## 7. まとめ

### 最重要ファインディング

1. **音声メモ/文字起こしカテゴリは「寒色の海」**: 青・紺・ダーク系で完全に埋め尽くされている
2. **暖色 (#E8956A) は完全な空白ゾーン**: 調査対象17アプリ中、ピーチ/テラコッタ系は皆無
3. **「風」モチーフも完全な空白**: 波形・マイク・文字ロゴが主流で、自然現象をモチーフにした音声系アプリは存在しない
4. **Soyokaは「色」と「モチーフ」の両軸で独自ポジションを確立できる**

### アクション推奨

1. **ブランドカラー #E8956A を堅持**: 競合との差別化の最大の武器
2. **そよ風モチーフの採用**: 名前との連想 + 音声波形のダブルミーニング
3. **Liquid Glass 対応の3レイヤーアイコンを早期設計**: iOS 26 で先行者優位
4. **A/Bテスト準備**: App Store のアイコンバリアントテストで、そよ風モチーフ vs 抽象暖色グラデーション を検証
