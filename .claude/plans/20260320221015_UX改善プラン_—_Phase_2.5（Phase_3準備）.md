# UX改善プラン — Phase 2.5（Phase 3準備）

## Context

Phase 2完了後のUXレビューで6つの体験上の問題を発見。Phase 3（AI処理実装）に入る前にこれらを解決し、設計書も同時に更新する。

## 解決する6つのUX問題

1. **録音完了→メモ詳細に直接遷移**（現状: メモ一覧に飛ばされる）
2. **AI処理はバックグラウンド+通知**（現状: 未実装）
3. **無料AI回数を月15回に引き上げ**（現状の設計: 月5回）
4. **感情分析をオプトイン**（現状の設計: デフォルトON）
5. **Sheet→NavigationDestinationに変更**（設計書準拠）
6. **録音完了画面の追加**（現状: なし）

## 実装順序（依存関係考慮）

```
Step 1: 問題5 — Sheet→NavigationDestination（基盤変更、他の前提）
Step 2: 問題6 — 録音完了画面追加
Step 3: 問題1 — 録音完了→メモ詳細遷移
Step 4: 問題4 — 感情分析オプトイン（設計書+コード）
Step 5: 問題3 — 月15回引き上げ（設計書のみ）
Step 6: 問題2 — AI処理バックグラウンド基盤（キュー+UI枠組み）
```
